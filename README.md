# 📡 Network Packet Sniffer using Python (Scapy)

🚀 A Cybersecurity & Networking project that captures and analyzes **live network traffic packets** using Python and Scapy.

---

## 📌 Project Overview

This project is a **real-time packet analyzer** that monitors network traffic and extracts useful information such as:

* Source & Destination IP addresses
* Protocol types (TCP, UDP, ICMP)
* Port numbers
* Packet payload (data)

It helps in understanding how data flows through a network and the structure of packets.

---

## 🎯 Objectives

* Capture live network packets
* Analyze packet structure and protocols
* Understand TCP/IP communication
* Build a basic network monitoring tool

---

## 🧠 Tech Stack

* 🐍 Python
* 📡 Scapy
* 🌐 Networking Protocols (TCP/IP)

---

## ⚙️ Workflow

1. 📥 Capture packets using Scapy
2. 🔍 Identify protocol layers (IP, TCP, UDP, ICMP)
3. 🌐 Extract source & destination IP
4. 🔢 Extract port numbers
5. 📦 Analyze payload data
6. 📊 Display structured output

## 🚀 How to Run

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/packet-sniffer.git
cd packet-sniffer
```

### 2️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Run the project

```bash
python main.py
```

> ⚠️ Note: Run with administrator/root privileges for packet capturing.
